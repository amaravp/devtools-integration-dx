# Alexa+ Integration Experience

A developer onboarding flow visualizing the jobs-to-be-done across 
three integration types — API, MCP, and Agent Relay — through the 
Alexa+ Workspace.

Built as a working prototype to support case study presentation and 
portfolio documentation.

## What's in here
- index.html — fully interactive flowchart, no dependencies beyond 
  Google Fonts and Material Icons
